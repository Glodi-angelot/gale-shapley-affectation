# Application d'affectation universitaire – Gale-Shapley

Ce projet implémente l'algorithme de Gale-Shapley pour résoudre le problème des mariages stables, appliqué à l'affectation des étudiants aux universités.

## Objectif

Attribuer à chaque étudiant une université de manière stable en respectant les préférences des deux parties.

## Fonctionnalités

- Génération automatique des données
- Simulation réaliste des préférences
- Exécution de l’algorithme de Gale-Shapley
- Vérification de la stabilité
- Interface graphique avec PyQt5
- Export des résultats (CSV, JSON)

## Technologies utilisées

- Python (structures standards)
- PyQt5 (interface graphique)
